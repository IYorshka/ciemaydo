MP4 MERGER
MP4 Merger — десктопное приложение на Python с графическим интерфейсом (Tkinter) для работы с видео. Использует ffmpeg под капотом.
  • Merge Videos — слияние нескольких MP4 в один (stream copy или re-encode)
  • Images to Video — создание видео из последовательности изображений (с аудио)
  • Rotate Video — поворот видео на 90/180 градусов
  • Smart Merge — слияние видео с разными разрешениями (авто или ресемпл)
Requirements:
Python 3.7+ · ffmpeg (imageio-ffmpeg) · moviepy (опционально)
Install: pip install imageio-ffmpeg moviepy
